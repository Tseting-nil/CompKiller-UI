## 建立開關

```lua
local Toggle = NormalSection:AddToggle({
	Name = "開關",
	Flag = "Toggle_Example", -- 留空則不儲存到設定檔（所有元素皆適用）
	Default = false,
	Callback = print,
});
```

### 開關參數說明

| 參數 | 類型 | 必填 | 說明 |
|------|------|------|------|
| `Name` | string | ✓ | 開關顯示名稱 |
| `Default` | boolean | ✗ | 預設值（預設為 false） |
| `Flag` | string | ✗ | 用於設定檔儲存的標記 |
| `Risky` | boolean | ✗ | 是否為高風險功能（顯示紅色文字） |
| `ToggleColor` | Color3 | ✗ | 自訂開關開啟時的顏色（可選） |
| `Callback` | function | ✗ | 值改變時執行的函數 |

### 自訂顏色開關範例

```lua
local Toggle = NormalSection:AddToggle({
	Name = "自訂顏色開關",
	Flag = "Toggle_Custom_Color",
	Default = false,
	ToggleColor = Color3.fromRGB(255, 100, 100), -- 自訂開關開啟時的顏色
	Callback = print,
});
```

## 建立按鈕

```lua
NormalSection:AddButton({
	Name = "按鈕",
	Callback = function()
		print('點擊!')
	end,
})
```

### 按鈕參數說明

| 參數 | 類型 | 必填 | 說明 |
|------|------|------|------|
| `Name` | string | ✓ | 按鈕顯示名稱 |
| `Callback` | function | ✓ | 點擊時執行的函數 |
| `Color3` | Color3 | ✗ | 自訂按鈕文字顏色（可選） |
| `BackgroundColor3` | Color3 | ✗ | 自訂按鈕背景顏色（可選） |

### 自訂顏色按鈕範例

```lua
-- 自訂文字顏色
NormalSection:AddButton({
	Name = "紅色文字按鈕",
	Color3 = Color3.fromRGB(255, 100, 100),
	Callback = function()
		print('紅色文字按鈕被點擊!')
	end,
})

-- 自訂背景顏色
NormalSection:AddButton({
	Name = "綠色背景按鈕",
	BackgroundColor3 = Color3.fromRGB(50, 200, 100),
	Callback = function()
		print('綠色背景按鈕被點擊!')
	end,
})

-- 同時自訂文字和背景顏色
NormalSection:AddButton({
	Name = "完全自訂按鈕",
	Color3 = Color3.fromRGB(255, 255, 255),
	BackgroundColor3 = Color3.fromRGB(100, 50, 200),
	Callback = function()
		print('完全自訂按鈕被點擊!')
	end,
})
```

## 設定管理器

### 寫入設定

將當前數值寫入設定檔

```lua
ConfigManager:WriteConfig({
	Name = "設定名稱",
	Author = "建立者名稱"
})
```

### 讀取資訊

讀取設定檔資訊，例如：建立者名稱和建立日期

```lua
ConfigManager:ReadInfo("<設定名稱>")
```

### 載入設定

載入目錄中的設定檔

```lua
ConfigManager:LoadConfig("<設定名稱>")
```

### 刪除設定

刪除目錄中的設定檔

```lua
ConfigManager:DeleteConfig("<設定名稱>")
```

### 取得所有設定

取得目錄中的所有設定檔

```lua
ConfigManager:GetConfigs()
```

### 目錄路徑

設定檔目錄路徑

```lua
ConfigManager.Directory
```
